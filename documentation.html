Introducción

TaskFlow es una aplicación web para gestión de tareas construida con tecnologías frontend modernas (HTML5, CSS3, JavaScript) y que utiliza LocalStorage para persistencia de datos.

Tecnologías utilizadas

  • HTML5: Estructura semántica y accesibilidad
  • CSS3: Flexbox, Grid, Variables CSS y diseño responsivo
  • JavaScript ES6+: Clases, módulos y LocalStorage API
  • LocalStorage: Persistencia de datos en el navegador

Instalación

TaskFlow no requiere instalación compleja. Solo sigue estos pasos:

# Clona el repositorio
git clone https://github.com/tu-usuario/taskflow.git

# Navega al directorio
cd taskflow

# Abre index.html en tu navegador
# O utiliza un servidor local para mejor experiencia
python -m http.server 8000

Requisitos

  • Navegador moderno (Chrome 70+, Firefox 65+, Safari 12+, Edge 79+)
  • JavaScript habilitado
  • Conexión a internet (solo para fuentes e iconos si se usan CDN)

Uso básico

TaskFlow es intuitivo y fácil de usar. Aquí los conceptos básicos:

Crear una tarea

  1. Haz clic en "Nueva Tarea" en el menú lateral
  2. Completa el formulario con los detalles de tu tarea
  3. Haz clic en "Guardar Tarea"

Gestionar tareas

Desde la vista de "Todas las tareas" puedes:

  • Marcar tareas como completadas usando el checkbox
  • Editar tareas haciendo clic en el icono de lápiz
  • Eliminar tareas con el icono de papelera
  • Buscar y filtrar tareas según tus necesidades
Nota: Todos los datos se guardan automáticamente en tu navegador.

Funcionalidades Detalladas

Sistema de Filtrado

TaskFlow incluye un sistema de filtrado avanzado que permite:

  • Filtrar por estado (pendiente, en progreso, completada)
  • Filtrar por prioridad (alta, media, baja)
  • Búsqueda por texto en título y descripción
  • Filtrado por fecha (hoy, esta semana, este mes)

Vista Dashboard

El dashboard proporciona una visión general de tu productividad:

  • Resumen de tareas totales, completadas y pendientes
  • Gráficos de distribución por prioridad
  • Tareas recientes y próximas a vencer

Personalización

TaskFlow puede personalizarse fácilmente modificando las variables CSS:

:root {
    --primary: #3b82f6;
    --primary-dark: #2563eb;
    --primary-light: #93c5fd;
    --secondary: #64748b;
    /* ... más variables */
}

Puedes cambiar los colores, espaciados, bordes y más mediante estas variables.

API Reference

TaskFlow expone una API simple a través de la clase TaskManager:

Métodos principales

constructor()

Inicializa el gestor de tareas y carga los datos existentes.

saveTask()

Guarda una tarea nueva o actualiza una existente.

deleteTask(id)

Elimina una tarea por su ID.

getFilteredTasks()

Devuelve las tareas filtradas según los criterios actuales.

Preguntas Frecuentes

¿Dónde se almacenan mis tareas?

Tus tareas se almacenan localmente en tu navegador usando LocalStorage. Esto significa que tus datos permanecen en tu dispositivo.

¿Puedo usar TaskFlow en múltiples dispositivos?

No directamente, ya que los datos se almacenan localmente. Sin embargo, puedes exportar tus tareas desde el código o implementar una sincronización con un backend.

¿Cómo puedo contribuir al proyecto?

Puedes hacer fork del repositorio en GitHub, realizar tus mejoras y enviar un pull request. Toda contribución es bienvenida.
